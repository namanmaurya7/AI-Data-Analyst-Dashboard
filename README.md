# 🚀 AI Data Analyst Dashboard

An AI-powered full-stack web application that allows users to upload CSV files, ask questions in natural language, and get instant insights with charts and summaries.

---

## 🧠 Features

- 📁 Upload CSV files
- 💬 Ask questions in natural language
- 🤖 AI converts query → structured format
- 📊 Visualizations (Bar Chart & Line Chart)
- 💡 Smart insights generation
- 💬 Chat-like UI interface

---

## 🛠️ Tech Stack

### Frontend

- React.js (Vite)
- Tailwind CSS
- Recharts

### Backend

- Node.js
- Express.js

### AI Integration

openrouterapi

### Data Processing

- Custom utility functions (filter, group, average)
